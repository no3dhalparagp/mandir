# User Guide: Phase 2 Enhancements

Welcome! This guide explains the new features and improvements added in Phase 2 of the Mandir Trust Accounting System.

## What's New?

### 1. Better Detail Pages

Previously, you could only see a list of items. Now you can click "View" to see complete details:

#### Devotees
- Full profile with all contact information
- Personal details (date of birth, anniversary)
- Family information
- Total donations and pujas in one place
- Special notes about the devotee

#### Puja Requests
- Complete puja request information
- Assigned priest details
- Can update status directly from the page
- Cost tracking (estimated vs. actual)
- Special requests visible
- Easy access to change puja status

#### Donations
- Full donation receipt information
- Donor details and donation method
- Receipt number and tracking
- Acknowledgment status
- Can print receipt for records
- Payment method details (cheque, bank transfer)
- In-kind donation details

### 2. Better Error Messages

When something goes wrong, you'll now see a clear error message in a red box with an icon, instead of just red text. This makes it easier to understand what happened.

### 3. Improved Tables

All list tables now have:
- **Hover effects**: Rows highlight when you move your mouse over them
- **View buttons**: Click to see full details
- **Better formatting**: Amounts shown as currency, dates formatted nicely
- **Status badges**: Color-coded to show status at a glance

## How to Use the New Features

### Viewing Devotee Details

1. Go to **Dashboard → Devotees**
2. Find the devotee you want to view
3. Click the **View** button in the rightmost column
4. See all their information on one page
5. Click **Back** to return to the devotees list

**What you'll see**:
- Profile card with status and joining date
- Contact information (email, phone, address)
- Personal details
- Total donations and puja requests

### Viewing Puja Request Details

1. Go to **Dashboard → Puja Requests**
2. Click the **View** button for the request you want to see
3. See complete request information
4. (Optional) Update the status using the dropdown
5. Click **Back** to return

**What you can do**:
- See full puja details
- See who requested it and when
- See cost information
- Change status from Pending → Confirmed → Completed → Cancelled
- See special requests and notes

### Viewing Donation Details

1. Go to **Dashboard → Donations** (under Mandir section)
2. Click the **View** button for the donation
3. See complete donation information
4. (Optional) Click **Print** to print the receipt
5. Click **Back** to return

**What you'll see**:
- Donation amount and purpose
- Donor name and donation date
- Receipt number
- Acknowledgment status
- Payment method (cheque, bank transfer, etc.)
- Any notes about the donation

## Color-Coding Guide

### Status Badges

**Blue** = Active/Confirmed/Posted
- Item is active and current

**Gray** = Lifetime/Completed/Secondary
- Item is completed or has special status

**Red** = Suspended/Cancelled
- Item is not active

**Yellow** = Pending/Draft
- Item is waiting for action

## New Keyboard Shortcuts

You can use these keyboard shortcuts for faster navigation:

- **Alt + D**: Go to Devotees
- **Alt + P**: Go to Puja Requests
- **Alt + N**: Go to Donations
- **Print**: Print current page (Ctrl+P or Cmd+P)

## Tips & Tricks

### Sorting & Filtering

- Use the dropdown menus on list pages to filter by status
- Tables show information in order of most recent first
- Click column headers to sort (if available)

### Mobile Use

- Swipe right/left on list tables to see all columns
- Buttons are sized for touch screens
- Detail pages stack for better mobile viewing

### Printing

- On donation detail pages, click "Print" or use Ctrl+P
- Your browser's print dialog will open
- Select "Save as PDF" to email donations receipts
- Select your printer to print physical copies

## Common Tasks

### I want to track a donation through its lifecycle
1. Go to Donations list
2. Click View on the donation
3. See when it was received
4. See acknowledgment status
5. Print receipt if needed

### I want to follow up on a puja request
1. Go to Puja Requests
2. Click View on the request
3. Check status and cost
4. Update status when puja is completed
5. See special requests the devotee made

### I want to maintain devotee records
1. Go to Devotees
2. Click View on the devotee
3. See all their information
4. See their donation history
5. See puja requests from them

## Troubleshooting

### "View" button doesn't work
- Make sure you have internet connection
- Try refreshing the page
- Check that you're logged in
- Contact support if problem persists

### Print dialog doesn't appear
- Make sure pop-ups aren't blocked
- Try using Ctrl+P or Cmd+P shortcut
- Check that JavaScript is enabled
- Try a different browser

### Data seems outdated
- Refresh the page (F5 or Cmd+R)
- Clear browser cache if data still shows old info
- Log out and log back in

### Status update didn't work
- Make sure internet connection is good
- Try updating again
- If error appears, note the error message
- Contact support with error details
